# Minimal Cafe Website (HTML & CSS)

This is a simple, two-page cafe website built using only HTML and CSS.  
It is a basic informational website designed to present a cafe concept and menu layout.

The project focuses on layout structure, styling, and basic UI design without using any JavaScript or frameworks.
